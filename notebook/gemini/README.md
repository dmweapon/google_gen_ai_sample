# Gemini Pro 
Gemini 는 Multi Modality 기반에서 학습된 모델입니다. 
Gemini Pro 내에는 아래와 같이 두가지 모델이 있으며, 각각 Text, Image, Video를 처리하는 목적으로 사용됩니다.  
    - Gemini Pro Text : 주로 Text 기반의 처리를 하는 Model.
    - Gemini Pro Vision : Image, Video 를 분석하는데 사용되는 모델.
